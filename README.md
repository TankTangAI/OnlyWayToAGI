## 1.2 自扩展记忆
不要人为决定哪些是该保存的
人的记忆还有一个特点就是“相关性”，也就是所谓的“联想”，比如我一看到“本田思域”这款车，我就想到我的一个大学室友，因为这个大学室友买的就是这款车，然后又不自觉的想到大学的其他室友，以及当年一起玩耍的情节……，这就是人类的“联想”本能，这一本能也是由于人的记忆是通过神经元之间的链接所形成的，而且人脑是一个乐此不疲的比较器，永远在把从外界接收到的刺激信号与之前的记忆进行匹配，如果完全匹配或者部分匹配，则会激活与之匹配的记忆区域（这种匹配可能会有很多层级，有点蝴蝶效应的意思），并且把新的刺激存储在这一区域附近或者与之连接处，表现出来就是会联想到之前与之关联的事情，比如事情A存储在大脑的m区域的1.2.3.4.5.6.7.8神经元之间的连接中，而事情B与事情A有关联，那么在大脑接收事情B的时候就会自动关联到事情A上，表现出来就是当你接收到事情B的时候你就会自动调取事情A，同时把事情B存储在A附近并与之关联。
